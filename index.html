HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • use only one h1 element
  • divs are block elements
  • class: give elements a class to be used with CSS
  • id: can only be used once , use with CSS & javaScript
  • spans are inline elements
  • tables are only for tabular data
  • forms have inputs with labels and submit buttons
  • inputs and submit buttons can have id's witch can be used with javascript
  • a is anchor element used to make hyperlinks
  • link: to link to CSS styles
  • script src: to link javascript to html
  • img/video/audio, src is to input media into html
  • use controls for video and audio
  • em emphasises content
  • br is a line break

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • :root; CSS pseudo-class matches the root element of a tree representing the documen example --color-primary: red;
  • grid, Layout Module offers a grid-based layout system, with rows and columns; example .container { display: grid; grid-template: repeat(4,100px)/repeat(4,100px); is short hand for grid template-rows/columns border: 3px solid lightslategray; height: 100vh;} can also use grid-template-rows: repeat(3, 100px); grid-template-columns: repeat(2, 100px);
  • :hover; pseudo-class generally triggered when the user hovers over an element with the cursor (mouse pointer).
  • :focus; generally triggered when the user clicks or taps on an element or selects it with the keyboard's Tab key.
  • select multiple classes with commma ,; example .header, .title
  • flexbox; display flex then can select flex direction row or colum
  • justify content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).
  • align content ; property aligns the flexible container's the items vertically.
  • media queries;define different style rules for different media types. example @media screen and (min-width: 680px) { .container { grid-template-rows: 4; } }
  • transform,property lets you rotate, scale, skew, or translate an element; example .box:hover { transform: rotate(45deg); }
  • CSS transitions allows you to change property values smoothly, over a given duration; example .box {transition: width 2s; } .box:hover { transform: rotate(45deg); }

Git

  • git clone repositoryName
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: checks what branch we are currently on
  • git add * stages new files and modifications, without deletions on the current directory and its subdirectories
  • git add -A: stages all changes
  • Git commit -m “message”
  • Git pull origin main
  • Git push: example feature/starter-code to your current branch not the main branch
  • In Github click on compare and pull request Give the pull request (PR) a descriptive title, like “add starter code for prework project”
  • select the "Create Pull Request" button: request a review first
  • Git checkout main to revert to the main branch
